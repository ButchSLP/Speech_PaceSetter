# Speech_PaceSetter
App for speech therapists to target speech fluency, speech rate, and speech clarity.  Client/student will match set pace during a reading passage.

This app displays a reading passage of which the words are highlighted at a set pace so that the user can practice speaking at a conversational pace.  The therapist can set the minimum and maximum rate of speech and the option of five different reading passages.  The student/client will read the passage while trying to keep the same pace as the highlighted word.
